# Node Learning

Express.js Tutorial: Build RESTful APIs with Node and Express | Mosh

[https://youtu.be/pKd0Rpw7O48](https://youtu.be/pKd0Rpw7O48])

### Initialize Node Project

```
npm init
```

### Install express

```
npm i express
```

### Install nodemon

```
npm install -g nodemon
```

### Install Joi

```
npm install joi
```

### To run
```
nodemon index.js
```
