Express.js Tutorial: Build RESTful APIs with Node and Express | Mosh

### Initialize a Node Project

```
npm init
```

### Install express

```
npm i express
```

### Install nodemon

```
npm i -g nodemon
```

### Install mongoose

```
npm i mongoose
```

### Install Joi

```
npm i joi
```

### To run

```
nodemon index.js
```
