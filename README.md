### Initialize a Node project

```
npm install npm@latest -g
```

```
npm init
```

### Install express

```
npm i express
```

### Install nodemon

```
npm i -g nodemon --no-optional
```

### Install mongoose

```
npm i mongoose
```

### Install Joi

```
npm i joi
```

### To run

```
nodemon index.js
```

### Robomongo
A free mongo client tool, [download](https://robomongo.org/download).
